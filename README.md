# Hosale Real Estate - Premium Property Portal

Hosale is a modern, premium, and fully responsive real estate web application built with **React** and **Vite**. The application features glassmorphic design elements, smooth scroll animations, dynamic counters, swiper-based carousels, category filtering, interactive forms, and dedicated authentication flows.

---

## 🚀 Live Demo & Repository
- **Repository:** [https://github.com/sndshx/hosale-property-portal](https://github.com/sndshx/hosale-property-portal)
- **Tech Stack:** React, Vite, React Router DOM, Swiper, ScrollReveal, Vanilla CSS

---

## ✨ Features

- **🏠 Interactive Homepage:**
  - Modern hero section with custom search filters (Location, Property Type, Price Range).
  - Floating glassmorphic cards and award badges.
  - Interactive counters using `IntersectionObserver` for user engagement.
- **📁 Multi-Page Routing:** Fully configured routes using `react-router-dom`:
  - **Home:** Main landing page with all highlights.
  - **About:** Mission statement, core values, and meets the team sections.
  - **Properties:** Grid layout with dynamic Category Filters (All, House, Villa, Apartment).
  - **Blog:** Curated article grid detailing industry insights.
  - **Contact:** Functional interactive contact form with dynamic success states.
- **🔒 Authentication Pages:** Clean, split-screen Login and Registration pages that auto-hide global layout navigation elements (Navbar & Footer).
- **💫 Dynamic Components:**
  - Responsive **Navbar** with active links and mobile toggle overlay.
  - **Swiper-powered Property Carousel** with autoplay and paginated navigation.
  - Custom scroll animations powered by **ScrollReveal**.
  - **ScrollToTop** helper ensuring smooth scroll resets on route changes.

## 🎨 Theme & Styling System

The application is styled with customized CSS variables for a premium brand feel:
- **Primary Color:** `#e0ebed` (Soft sky background tint)
- **Accent Dark Green:** `#25453a` (Forest green main theme)
- **Accent Light Green:** `#3a6b57` (Interactive hover state)
- **Typography:** Modern sans-serif fonts with strict hierarchy.
- **Effects:** Glassmorphic navigation headers, gradient borders, dynamic card hovers, and animated badges.

## 🛠️ Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/sndshx/hosale-property-portal.git
   cd hosale-property-portal
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```
   *The application will automatically open in your default browser at `http://localhost:5173/`.*

4. **Build for production:**
   ```bash
   npm run build
   ```
